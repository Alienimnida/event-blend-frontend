# Eventblend Frontend

## Setup

### Install dependencies

```
pnpm i
```

### Add env variables

```
cp .env.example .env
```

### Start dev server

```
pnpm run dev
```

### Open

http://localhost:5173/

## Tech stack

- Typescript
- Qwik
- Qwikcity
- Qwik UI
- Tailwindcss
- Valibot
- Wretch
- Modular forms

**Please go through CONTRIBUTING.md file before start contribution and code changes**.
